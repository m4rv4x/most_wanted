# Most_Wanted (beta)
## Simple bash scripts to find ip and country of bad guys from logs

## Run it as Root !

- `./apache2.sh`
- `./fail2ban.sh`
- `./lighttpd.sh`
- `./openvpnas.sh`
- `./ufw.sh`
